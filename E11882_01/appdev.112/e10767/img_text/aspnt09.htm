This screen shot shows the Data Source Configuration wizard. You are requested to choose a Data Source type, of the following Access Database, Database, LINQ, Object, Site Map and XML file. Database is highlight. When this is chosen, under the list it says "Connect to any SQL database supported by ADO.NET, such as Microsoft SQL Server, Oracle, or OLEDB." After this is a field for entering an ID for the data source, which is HR. Finally, there are OK and Cancel buttons.
