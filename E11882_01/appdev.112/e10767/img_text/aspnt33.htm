This screen shot shows the Add New Access Rule section. Under it are several parts. First there is a Select directory for this rule: This shows the tree view of WebSite1 expanded to show the App_Data folder. The second part is Rule applies to: with options to choose Role (includes a list) and User (includes an entry field). After that there is a link to Search for users. Below that are options to choose All users or Anonymous users. The last part is Permission: with option for All and Deny (selected). Last is the OK button.
