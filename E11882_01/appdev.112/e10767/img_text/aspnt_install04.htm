The window shows the Schema: SYS from a list, Object type: USER from a list, and ObjectName: ANONYMOUS, which is indicated to be disabled, also from a list.

There is a User/Role section. User is selected and ASPNET_DB_USER is selected from alist. This is followed by the Privileges sections. There are two lists, one is Available Privileges which contains all the avaiable privileges except those on the right which are the Granted Privileges. There are arrows between the lists for clicking from one list to another.

At the end, are the following buttons: Preview SQL, OK, Cancel, Apply, and Help.
