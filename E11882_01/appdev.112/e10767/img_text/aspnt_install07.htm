This screen shot shows the Run SQL*Plus Script feature. It has groups for File Information, with a field for specifying the full path of the SQL*Plus script with a browse button, Connection Information where the database connection information is specified as ASPNET_DB_USER.ORCL, with a New Connection button provided for browsing, and finally the Connection String section currently showing DATA SOURCE=ORCL;PERSIS SECURITY INFO=False;USER. This is followed by a read-only user input area.There is an executing script bar and Run and Cancel buttons.
