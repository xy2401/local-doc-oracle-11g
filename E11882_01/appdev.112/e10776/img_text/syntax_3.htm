This is a text description of syntax3.gif. This image shows a syntax diagram, which shows the following available options for 8-BIT and 16-BIT operator values:

  • 8-BIT

    • DRCT GRAY

    • DRCT GRAY SCALE

    • GRAY

    • GRAY SCALE

  • 16-BIT

    • DRCT GRAY A

    • DRCT GRAY SCALE A

    • GRAY A

    • GRAY SCALE A

    In the preceding lists, the spelling G-R-A-Y can substituted with the spelling G-R-E-Y.
