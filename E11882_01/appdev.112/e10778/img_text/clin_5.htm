This is a screen capture of the Image Summary area on the DICOM Image Archive Home page.

This screen capture shows a list of fields containing the summary data for thumbnail image 1 (Image Summary: 1) of the six images that are installed with the sample application. Some fields are blank, while others contain data such as patient name, study date, and modality. (The full list of fields is described in Section 6.2.2.1.)
