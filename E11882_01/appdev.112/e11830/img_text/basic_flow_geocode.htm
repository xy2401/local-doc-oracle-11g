This is a text description of Figure 11-1, "Basic Flow of Action with the Spatial Geocoding Service". It illustrates the basic flow of action with the Spatial geocoding service, as explained in text before and after the illustration.

A geocoding request includes one or more input addresses. Addresses can be formatted or unformatted.

A geocoding response includes a geocoded address for each input address. Each geocoded address includes longitude and latitude coordinate values, and other information.
