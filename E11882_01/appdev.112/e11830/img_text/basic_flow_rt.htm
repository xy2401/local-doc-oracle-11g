This is a text description of Figure 13-1, "Basic Flow of Action with the Spatial Routing Engine". It illustrates the basic flow of action with the Spatial routing engine, as explained in text before the illustration.

A route request includes preferences, the start location, and the end location. A route response includes route information, and segment information for each segment.

A batch route request includes preferences, the start location, and one or more end locations. A batch route response includes route information for each route.
