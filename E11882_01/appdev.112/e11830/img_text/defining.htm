This is a text description of Figure 7-8, "Defining a Geometric Segment". It illustrates three different ways of defining a geometric segment: (a) assigning no measure values (all measure values null); (b) assigning only the start and end measure values, but no measure values for intervening points; and (c) assigning measure values to the start and end points and all intervening shape points.
