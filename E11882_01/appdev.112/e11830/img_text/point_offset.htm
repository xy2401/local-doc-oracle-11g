This is a text description of Figure 7-15, "Locating a Point Along a Segment with a Measure and an Offset". It illustrates locating a point along a segment with a measure and an offset. The offset of the point to be located is positive if it is to the left along the segment direction and negative if it is to the right along the segment direction.
