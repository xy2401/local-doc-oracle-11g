This is a text description of Figure 2-9, "Geometry with Type 0 (Zero) Element". It illustrates a type zero element, as described in text before and after the illustration. The illustration refers to coordinates as simply x1, y1, x2, y2, and so on, with no reference to a specific spatial dimension. However, the example after the explanation of the illustration (at the end of this section) inserts a geometry with a type 0 element into the spatial dimension for the cola market geometry examples that are used throughout this guide.
