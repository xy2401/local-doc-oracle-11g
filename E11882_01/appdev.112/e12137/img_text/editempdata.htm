This image shows the edit.jsp page in a browser. This page contains, apart from formatted static text (heading, rule, and so on), an HTML table with two columns. The left column contains labels for the employee data that is retrieved from the database, such as First Name, Last Name, Email, and so on. The right-hand side column contains text fields populated with data for an employee. At the bottom of the page is an update button to submit any changes made to the data in the text fields.
