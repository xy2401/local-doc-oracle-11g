This diagram shows a disk icon labeled "Oracle XML DB". The icon connects below it to four boxes with the following names: "Unifies Data and Content", "Faster Storage and Retrieval of Complex XML Documents", "Helps Integrate Applications", and "Also Handles non-XML Data with XMLType Views".
