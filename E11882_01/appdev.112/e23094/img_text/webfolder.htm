This is an image of an Internet Explorer browser Web Folder Add Network Place Wizard, used to create a Web folder in Microsoft Windows. Using this you can access Repository resources through the Repository's WebDAV support. Under "Type the location of the Network Place" (you need to access), you can enter the location or select the Browse button to navigate to the location. Click Next to access the location entered.
