Shows the Mining Model components. This includes an area showing graphs in BIEE Reports that use the mining model. There is a box showing the components of the mining model in the Oracle Database. Within the ORDM_SYS schema are the following components: ORMD Main Model, Mining ETL, Mining Source Views, Mining Package, Mining Support Tables, and Mining Result Tables.
