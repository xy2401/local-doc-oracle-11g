Shows the Oracle Retail Data Model entity types and the path for populating tables. Includes: Source ETL to populate Base, Reference, and Lookup tables. These tables use the Derived Intra-ETL to populate the Derived tables. Then, the Aggregate Intra-ETL populates the Aggregate tables from the Derived tables. Some of the population of the Aggregate tables is shown as being directly from the Base, Reference and Lookup tables.
