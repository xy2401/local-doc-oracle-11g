Shows the New Dashboard dialog. This includes the fields: Name, Description, Location, and Content. Content includes radio buttons: Add content now (selected), and Add content later (create empty dahsboard). The dialog includes the buttons: Help, OK, and Cancel. The field Name, has the value: Category Manager. The field Description, has the value Category Manager. The Location field shows: /Shared Folders/Category Manager/Dahsboards.
