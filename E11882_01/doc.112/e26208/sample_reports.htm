11 Oracle Airlines Data Model Sample Reports

This chapter provides Oracle Airlines Data Model sample reports and includes the following sections:

  • Agent Performance Analysis

  • Booking Analysis

  • Channel Performance Analysis

  • Revenue Analysis

  • Route Analysis Reports

  • Call Center Performance Analysis

Note:

The reports and dashboards shown in the examples in this chapter and delivered with Oracle Airlines Data Model are provided only for demonstration purposes. These sample reports and dashboards are not supported by Oracle.
