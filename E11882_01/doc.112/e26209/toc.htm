Known Issues

This release has the following known issue:

  • Oracle Airlines Data Model Schema Must Reside on the Same Disk as the Database

Oracle Airlines Data Model Schema Must Reside on the Same Disk as the Database

The Oracle Airlines Data Model schema needs to reside on the same disk as the Database. When the user chooses a different location, the installation fails.

This issue is documented in Bug 13453903.

Workaround

When installing Oracle Airlines Data Model, choose the same oradata location as the Database for the installation of the Oracle Airlines Data Model schema.
