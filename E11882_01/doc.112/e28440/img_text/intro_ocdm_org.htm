Shows the architecture of Oracle Communications Data Model. This includes an outer layer of "Security". There is a container with Data Sources that contains the Staging Layer, the Foundation Layer, and the Access and Performance Layer. This area includes the ETL, Messaging, and Metadata. The flow is from Data Sources, one of: COTS Operational systems, Unstructured data, or syndicated external, or Master Data to the staging layer (including temporary loading structures and rejected data). Then to the foundation layer, then to the Access and Performance Layer. The figure also shows BI Abstraction and Query Generation, Performance Management Apps, and Advanced Analysis Tools, Alerts, Dashboards and MD-Analysis Reports, and Web Service.
