Oracle R Enterprise client components are installed in a directory that you create. In this example, the directory is called myoreclient. Subdirectories that contain the Oracle R Enterprise client components are:

client -- contains the Oracle R Enterprise client packages

instantclient_12_ -- contains Oracle Instant Client for Oracle Database 12.1

supporting -- contains the Oracle R Enterprise client supporting packages

The directories were extracted from these zip files which are also in the the client installation directory myoreclient:

instantclient-basic-windows.x64-12.1.0.1.zip

instantclient-sdk-windows.x64-12.1.0.1.zip

ore-client-win-x86.64-1.3.1.zip

ore-supporting-win-x86_64-1.3.1.zip
