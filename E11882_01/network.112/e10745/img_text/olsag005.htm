This illustration presents a matrix of levels and compartments. Each level may contain all compartments, some compartments or no compartments. For example, the HIGHLY_SENSITIVE (HS) level in the figure contains compartments FINCL, CHEM, and OP; the SENSITIVE (S) level contains compartments FINCL and OP; the PUBLIC (P) level contains only the compartment OP.
