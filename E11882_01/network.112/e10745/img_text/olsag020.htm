This illustration describes label tags in a distributed database. There are two lists, one for the EASTERN_REGION database and one for the WESTERN_REGION database. Each database has an identical list of character string labels, namely: S:A, C:A, and U. However, the label tags associated with the character strings are different in the two databases. In the EASTERN_REGION database the label tags are 3001, 2001, and 10, respectively. In the WESTERN_REGION database the label tags are 11, 6, and 5, respectively.
