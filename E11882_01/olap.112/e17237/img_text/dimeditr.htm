Results of the DimEditorPlugin: This figure shows the navigation tree in Analytic Workspace Manager with the CHANNEL dimension selected in the MyDims folder that is created by the aw.xml document. The property inspector displays the output of the DimEditorPlugin. The plug-in displays the name of the dimension, which is CHANNEL, and the name of the short description attribute, which appears as Short Description. In an editable text field it displays the value of the short description attribute, which is Channel.
