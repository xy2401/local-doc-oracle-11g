Results of the MyLevels <AWMNode> in dimension.xml: This figure shows the navigation tree in Analytic Workspace Manager with the MyLevels folder selected. The folder is created by an <AWMNode> in the dimension.xml document. In the folder are the levels that are the result of the SQL statement in the <AWMNode>.

The report shows the query, select level_name from user_cube_dim_levels where dimension_name = ?, and the query results, which is a column with the heading LEVEL_NAME and the list of levels, which are ITEM, FAMILY, CLASS, and TOTAl.
