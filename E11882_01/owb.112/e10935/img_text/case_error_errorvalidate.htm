This screenshot displays the Results tab in the Log window. At the top of the tab are the following icons, from left to right: Show Warnings, Show Errors, Go To Source, View Monitor, View Script, and Save Script As. Below these icons is a top-level node that indicates the status of the validation. In this image, this node is expanded and displays Completed (1 objects have errors, 0 have warnings). Under this node is the CUSTOMERS node, which is expanded to display the Validation node. Under the Validation node is the validation message VLD-0215.
