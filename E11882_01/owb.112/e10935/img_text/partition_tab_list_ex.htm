This image displays the grid in the Partitions tab that is used to define a list partition based on a single key column. The grid contains the following columns, from left to right: Type, Key Columns, Partitions, and Values.

The first row in the grid displays List, which represents the type of partition created. One level below this row are the following rows, from top to bottom: Partition Key and Partitions. Below the grid are the following buttons, from left to right: Add, Add Subpartition, Add Hash Count, and Delete. The Add Subpartition and Add Hash Count buttons are disabled.
