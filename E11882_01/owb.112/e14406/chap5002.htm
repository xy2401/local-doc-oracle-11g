95/513

OMBALTER USER

Purpose

To alter properties of a Warehouse Builder user.

Prerequisites

Must be connected to a workspace on OWB repository.

Syntax

alterUserCommand =  OMBALTER ( USER "QUOTED_STRING" ( SET 
          "setPropertiesClause" ) ) 
     setPropertiesClause =  PROPERTIES "(" "propertyNameList" ")" VALUES "(" 
          "propertyValueList" ")"
     propertyNameList =  "UNQUOTED_STRING" { "," "UNQUOTED_STRING" }
     propertyValueList =  "propertyValue" { "," "propertyValue" }
     propertyValue =  ( "QUOTED_STRING" | "INTEGER_LITERAL" | 
          "FLOATING_POINT_LITERAL" )

Parameters

alterUserCommand

This clause alters a Warehouse Builder user.

setPropertiesClause

Used to set properties of a Warehouse Builder user. Valid properties are shown below.
