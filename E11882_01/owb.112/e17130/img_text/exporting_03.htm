The Metadata Export dialog box contains:

  • Table of objects selected for export, with columns Object Name and Object Type

  • Textbox for Annotations

  • File Name textbox with associated Browse button

  • Log File textbox with associated Browse button

  • Checkbox for Export all object dependencies

  • Advanced button

  • Help button

  • Export button

  • Cancel button
