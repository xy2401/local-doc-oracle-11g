The Metadata Import dialog box contains:

  • File Name textbox with associated Browse button

  • Log File textbox with associated Browse button

  • Object Selection section with radio buttons for one of Import all objects from file or Import selected objects from file

  • Import Option section with radio buttons for one of Create new metadata only, Update metadata replace existing objects and create new metadata, Merge metadata merge existing objects and create new metadata, or Replace existing objects only

  • Match by section with radio buttons for one of Universal Identifier or Names

  • Show Summary button

  • Advanced button

  • Help button

  • Import button

  • Cancel button
