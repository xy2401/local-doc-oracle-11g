Default Object Privileges appear in the Edit User window.

The information is presented on the main region of the window in the form of a clickable table, with columns Principal (listing various Roles and Users), FULL_CONTROL (checkbox), EDIT (checkbox), COMPILE (checkbox), and READ (checkbox).

The left region of the window provides navigation between the four options of the Edit User interface: Name, Roles, Default Object Privilege, and System Privilege.

The bottom of the window contains three buttons: Help, OK, and Cancel.
