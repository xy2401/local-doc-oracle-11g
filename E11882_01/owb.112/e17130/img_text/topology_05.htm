This graphic describes a Topology that uses the Control Center Agent; it uses 1 client and 4 server computers.

  1. The Oracle Database is installed on the Server 1 computer. The Oracle Database contains a Repository, a Target Schema, and the CCA Audit Schema.

    • The Repository contains at least one Workspace, which contains Audit Tables, TS1 (with Map, T1, and T2), SS1 (with T1, and T2), and SS2 (with T1, and T2).

    • The Target Schema contains a Map, T1 and T2.

    • The CCA Audit Schema contains Audit Tables.

  2. A local OC4J service is installed on Server 2; it contains the Control Center Agent (CCA), Map1a, and Map1b.

    • The Control Center Agent contains a Map, DS1, DS2, and DS3.

      • The Map controls the content of Map1a and Map1b.

      • DS1 executes on Source Schema 2 on Server 3.

      • DS2 executes on Source Schema 2 on Server 4.

      • DS3 executes on the Target Schema in Oracle Database on Server 1.

  3. A DB2 database in installed on Server 3. It contains Source Schema 1.

  4. A SQLServer database in installed on Server 4. It contains Source Schema 2.

  5. The Oracle Warehouse Builder Control Center Service is installed on the Server 1 computer; it deploys and executes on the Control Center Agent on Server 2.

  6. The Design Center is installed on the client computer, and it manages the design inside the Repository that is in the Oracle Database (on the Server 1 computer). It also deploys and executes on Oracle Warehouse Builder Control Center Service (on the Server 1 computer).

  7. The other Local OC4J service is installed on the client computer, and it contains the Browser Server. It reports on the Repository in the Oracle Database (on the Server 1 computer), it reports on the Audit Tables that are in the Workspace of the Repository in the same Oracle Database, and it reports on the Audit Tables that are in the in the CCA Audit Schema on the same Oracle Database. It also receives and updates Oracle Warehouse Builder Browser's Design Reports and Audit Reports (on the client computer).

  8. Oracle Warehouse Builder Browser is installed on the client computer, and it receives and updates Local OC4J service's Design Reports and Audit Reports (on the client computer).
