This illustration shows two failover scenarios.

One scenario uses multimaster replication. When the primary database fails, the fail-over database is another master site in the replication environment.

The other scenario shows an Oracle Real Application Clusters (Oracle RAC) environment. When one instance in the cluster fails, another instance in the cluster is used for failover protection.
