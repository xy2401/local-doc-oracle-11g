This illustration shows a flowchart for creating materialized views.

  1. Start.

  2. Set up the materialized view site.

  3. Create materialized view logs at the master site.

    Question: Type of materialized view creation? Choices: Individually or Deployment Template

  4. For individually created materialized views, connect to a remote materialized view site.

    For Deployment Template materialized view creation, create a deployment template.

  5. For individually created materialized views, create materialized views and end.

    For Deployment Template materialized view creation, package the template and distribute.

  6. For Deployment Template materialized view creation, instantiate the template at the materialized view site and end.
