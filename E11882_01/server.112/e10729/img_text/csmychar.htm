The image csmychar.gif shows the General tab page for MYCHARSET. From top to bottom, the text fields and their values are as follows:

  • Character Set Name (MYCHARSET)

  • Character Set ID (10001)

  • ISO Character Set ID (blank)

  • Base Character Set ID (blank)

The Show Existing Definitions button is at the bottom of the tab page. A toolbar is on the left side of the screen. The following menus are at the top of the screen: File, Edit, Tools, Help.
