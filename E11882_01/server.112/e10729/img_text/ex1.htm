The image ex1.gif shows the Canonical Rules dialog box. It contains a combo box that shows how 3 characters are decomposed. The column headings are:

An example of a decomposed character is given in the text following image.

  • Add (unavailable)

  • Modify (unavailable)

  • Delete (unavailable)

  • Search

An OK button appears in the lower corner of the dialog box.
