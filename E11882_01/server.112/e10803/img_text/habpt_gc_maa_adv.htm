The MAA Advisor page shows the "All Solutions" that contains a table with column headers for Outage Type, Oracle Solution, Recommendation Level, Target Name:Role, Configuration Status, and Benefits.

For some solutions, the third column contains the word "High" to show a strong recommendation for configuring Human Errors- Erroneous Transactions (with Flashback Query, Flashback Transaction, or Flashback Transaction. For Data Corruptions: Configure DB_ULTRA_SAFE Initialization Parameter.
