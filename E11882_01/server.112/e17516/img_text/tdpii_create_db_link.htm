This screenshot shows the Create Database Link page. At the top right-hand corner of the page are the buttons: Show SQL, Cancel, and OK. Below the buttons are the General and Connect As sections:

  • The General section includes:

    • A Name field

    • A Net Service Name field

    • A check box labeled Public - This database link is available to all users.

  • The Connect As section includes:

    • A Connected User option

    • A Current User option

    • A Fixed User option with Username, Password, and Confirm Password fields

The buttons at the top right-hand corner of the page are repeated at the bottom right-hand corner of the page.
