This figure illustrates the components in a Change Data Capture system that are of particular interest to the subscribers. The image is labeled "Staging Database: DW". Below that is a box labeled "Change Set: SH_SET". Within the Change Set box is a box on the left (outlined in dashes) labeled "Publication on SH.SALES". This dashed box contains the following list:

  • PROD_ID

  • CUST_ID

  • TIME_ID

  • PROMO_ID

  • QUANTITY_SOLD

  • AMOUNT_SOLD

Overlaying the dashed box, is a box with solid lines labeled "Subscriber View: spl_sales". This box includes a subset of the previous list:

  • PROD_ID

  • CUST_ID

  • TIME_ID

  • PROMO_ID

Below the box labeled "Publication on SH.SALES" is another box (outlined in dashes), labeled "Publication on SH.PROMOTIONS". This dashed box contains the following list:

  • PROMO_ID

  • PROMO_SUBCATEGORY

  • PROMO_CATEGORY

  • PROMO_END_DATE

  • PROMO_BEGIN_DATE

Overlaying the dashed box, is a box with solid lines labeled "Subscriber View: spl_promos". This box includes a subset of the previous list:

  • PROMO_ID

  • PROMO_SUBCATEGORY

  • PROMO_CATEGORY

There are arrows pointing from each of the subscriber view boxes to a box on the right labeled "Subscription: sales_promos_list".
