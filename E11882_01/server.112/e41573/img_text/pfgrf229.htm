This graphic shows a client process with two sessions. Each session submits the same SQL statement: SELECT department_id FROM departments. Each session retrieves the result set from the Client Result Cache. The Result Set contains the values 10,20,30,40, and so on. A vertical dotted line divides this part of the diagram, labeled Client, from the part labeled Server. A database icon is shown interacting with the cache, with the label "Keeps Consistent."
